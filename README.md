# 2048.vim
> 2048 in vim

[![](https://spacevim.org/img/build-with-SpaceVim.svg)](https://spacevim.org)

## Install

this plugin is based of SpaceVim's API, you can load the game layer, and run the game by <kbd>SPC g 2</kbd>.
